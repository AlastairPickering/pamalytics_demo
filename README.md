# PAMalytics validation demo

This repository contains a lightweight hosted demonstration of the PAMalytics validation workflow.

The demo opens directly into the Validation page using an example project. It is intended for peer review and for prospective users who want to try the validation interface before installing the full PAMalytics application.

## What is included

- PAMalytics validation page
- validation strategy selection
- spectrograms and audio playback
- manual validation controls
- preconfigured example project
- validated CSV export


## Notes

The demo is a fixed example workflow. The full PAMalytics application should be installed locally for real analyses.
